## Juego de ejemplo: Texas Hold'em para 5 jugadores

Se incluye el script `texas_holdem.py` que simula una ronda de Texas Hold'em con cinco jugadores. Para ejecutarlo:

```bash
python3 texas_holdem.py
```

Al correr el script se reparten las cartas, se muestran las cartas comunitarias y la mano de cada jugador.
